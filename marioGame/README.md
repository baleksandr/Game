# 🍄 Mario Platformer

Платформер в стилі Super Mario, створений на **PixiJS**.

## 🎮 Керування

| Клавіша | Дія |
|---------|-----|
| ← / A | Рух вліво |
| → / D | Рух вправо |
| ↑ / W / Space | Стрибок |
| P / Esc | Пауза |

## 🚀 Запуск

### Встановлення залежностей:
```bash
cd marioGame
npm install
```

### Запуск в режимі розробки:
```bash
npm run dev
```

### Збірка для продакшену:
```bash
npm run build
```

## 📁 Структура проекту

```
marioGame/
├── src/
│   ├── main.js          # Точка входу
│   ├── Game.js          # Головний клас гри
│   ├── Player.js        # Клас гравця
│   ├── Platform.js      # Платформи та блоки
│   ├── Coin.js          # Монети для збору
│   ├── Enemy.js         # Вороги (гумби)
│   ├── InputHandler.js  # Обробка введення
│   └── Background.js    # Фон та паралакс
├── styles/
│   └── style.css        # Стилі інтерфейсу
├── index.html           # Головна HTML сторінка
├── package.json         # Залежності npm
└── README.md            # Цей файл
```

## 🎯 Геймплей

- **Збирай монети** 🪙 - кожна монета дає 100 очок
- **Стрибай на ворогів** 👾 - знищуй гумб стрибаючи на них (200 очок)
- **Активуй блоки ?** ❓ - бий головою по блоках з питанням (50 очок)
- **Досягни прапора** 🚩 - пройди рівень до кінця

## ✨ Особливості

- Плавна фізика з гравітацією
- Система життів та очок
- Анімації гравця та ворогів
- Паралакс-ефект фону
- Адаптивне управління (клавіатура + мобільні кнопки)
- Красивий ретро-стиль інтерфейсу

## 🛠 Технології

- [PixiJS 8](https://pixijs.com/) - 2D WebGL рендерер
- [Vite](https://vitejs.dev/) - збірник та дев-сервер
- Ванільний JavaScript (ES6 модулі)
